# particle_transformer

## To run:

```bash
python particle_transformer.py
```

environment.yaml is provided.

Example output can be found in temp.txt (Created with: 'time python -u golden.py > temp.txt').

wget https://zenodo.org/record/6619768/files/JetClass_Pythia_train_100M_part0.tar?download=1 \

Change sub dataset's name if needed. Currently we are just using part0 and val part.
